  <action>
Create `scripts/smoke_torch.py` as a standalone executable Python script (with `if __name__ == "__main__"` guard) that validates ENV-01:

1. Import torch, assert `torch.cuda.is_available()` is True (exit 1 with message if not)
2. Get `torch.cuda.get_arch_list()` and assert `"sm_120"` is in the list
3. Get `torch.cuda.get_device_capability(0)` and assert major >= 12
4. Print device name via `torch.cuda.get_device_name(0)`
5. Run a GPU tensor smoke test: create two 1000x1000 random tensors on CUDA, matrix multiply, assert output shape is (1000, 1000)
6. Print summary: PyTorch version, CUDA version (`torch.version.cuda`), device name, capability, and "ENV-01 PASS"
7. Exit 0 on success, exit 1 on any assertion failure with descriptive error message

Use `sys.exit(1)` on failure, not just assert — so the master script can detect non-zero exit codes. Wrap in try/except to catch and print any unexpected errors before exiting.

**Audio fixture:** Create `scripts/test_audio/sample_16k.wav` — a real 16kHz mono PCM16 WAV file containing 3-5 seconds of clear English speech. Generate it programmatically using a short TTS utterance or, if no TTS is available in the dev environment, generate a synthetic spoken-like signal using scipy/numpy:
- Use `scipy.io.wavfile.write()` to create a valid WAV
- If scipy is not available, use the `wave` stdlib module
- The content must be a valid WAV that NeMo can load — at minimum a 3-second 16kHz mono file
- Preferred: use Kokoro (if installed) to generate a real speech clip of "The quick brown fox jumps over the lazy dog"
- Fallback: create a 3-second sine-wave-modulated signal that won't crash soundfile.read() — the ASR result may be empty/garbage but the script will still validate GPU execution path

**Smoke script:** Create `scripts/smoke_parakeet.py`:

1. FIRST LINE after imports-guard: `os.environ["NEMO_DISABLE_CUDA_GRAPHS"] = "1"` — MUST be before any NeMo import (this is the Blackwell requirement from RESEARCH.md)
2. Also set `os.environ.setdefault("TORCH_FORCE_NO_WEIGHTS_ONLY_LOAD", "1")` for PyTorch 2.6+ checkpoint compat
3. Import `nemo.collections.asr as nemo_asr`
4. Load model: `nemo_asr.models.ASRModel.from_pretrained("nvidia/parakeet-tdt-0.6b-v2")`
5. Call `model.eval()`
6. Load `scripts/test_audio/sample_16k.wav` via `soundfile.read()`, assert sr == 16000
7. Call `model.transcribe([audio], return_hypotheses=True)`
8. Extract text from result (handle tuple and list return formats, same as asr.py does)
9. Print the transcription text
10. Assert `len(text) > 0` (non-empty transcription proves GPU execution)
11. Print "ASR-01 PASS" on success
12. Use try/except with `sys.exit(1)` on failure, same pattern as smoke_torch.py

Add a VRAM report after model load using `torch.cuda.memory_allocated() / 1024**3` — this helps inform ENV-02 budget in Plan 02.
