            - If navigating: "Navigating to [destination]..." with progress indicator (LinearProgressIndicator if progress available)
            - If arrived: "Arrived at [destination]" with green checkmark icon
            - If failed: "Navigation failed" with red warning icon and retry suggestion
            - If no active nav: "Ready" or empty state
