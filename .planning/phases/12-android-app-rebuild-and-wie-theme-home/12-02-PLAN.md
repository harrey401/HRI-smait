  <action>
    Create the WebSocket data layer:

    1. **WebSocketEvent.kt** — Sealed class:
       ```
       sealed class WebSocketEvent {
         data class BinaryFrame(val bytes: ByteArray) : WebSocketEvent()
         data class JsonMessage(val type: String, val payload: String) : WebSocketEvent()
         object Connected : WebSocketEvent()
         data class Disconnected(val reason: String?) : WebSocketEvent()
       }
       ```

    2. **WebSocketRepository.kt**:
       - Constructor: `(client: OkHttpClient)` — injectable for testing
       - `private val _events = MutableSharedFlow<WebSocketEvent>(extraBufferCapacity = 64)`
       - `val events: SharedFlow<WebSocketEvent>` — public read-only
       - `private val _isConnected = MutableStateFlow(false)`
       - `val isConnected: StateFlow<Boolean>`
       - `fun connect(url: String)` — creates WebSocket via OkHttp3, registers WebSocketListener
       - `onMessage(ws, ByteString)` → emit `BinaryFrame(bytes.toByteArray())`
       - `onMessage(ws, String)` → parse JSON to extract "type" field, emit `JsonMessage(type, rawJson)`
       - `onOpen(ws)` → emit `Connected`, set isConnected=true
       - `onFailure(ws, t, response)` → emit `Disconnected(t.message)`, set isConnected=false, schedule reconnect with exponential backoff (1s, 2s, 4s, max 30s)
       - `onClosed(ws)` → emit `Disconnected("closed")`, set isConnected=false
       - `fun send(bytes: ByteArray)` → `webSocket?.send(bytes.toByteString())`
       - `fun send(json: String)` → `webSocket?.send(json)`
       - `fun disconnect()` → `webSocket?.close(1000, "Client disconnect")`
       - `private fun scheduleReconnect()` — uses coroutine delay with exponential backoff

       CRITICAL: Do NOT filter or transform binary frames in the repository. All bytes pass through as-is. The type byte (first byte) is preserved — consumers (ViewModels, AudioManagers) read it to determine handling. This matches the existing protocol where the first byte is the frame type.

       CRITICAL: For JSON messages, parse minimally — just extract the "type" field from the JSON object using `org.json.JSONObject(text).optString("type", "unknown")`. Pass the full raw JSON string as payload. Do NOT deserialize into domain objects here.

    3. **Tests** — Use MockWebServer from OkHttp3 test library, OR mock OkHttpClient and WebSocketListener to verify event emission. Use Turbine for Flow assertion. If MockWebServer is too heavy, test the event mapping logic in isolation: create the listener, call its methods directly, and verify SharedFlow emissions.
