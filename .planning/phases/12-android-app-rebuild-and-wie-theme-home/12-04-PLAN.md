    3. **VideoStreamManager.kt** — Manages continuous Camera2/CameraX video capture and sends 0x02 frames:
       - Constructor: `(wsRepo: WebSocketRepository)`
       - `fun start(context: Context)` — opens camera (prefer LENS_FACING_EXTERNAL for Jackie's USB camera, fallback to LENS_FACING_FRONT), creates ImageReader(320, 240, ImageFormat.YUV_420_888, 2), registers OnImageAvailableListener

       - Target ~10fps: throttle via timestamp check (skip frame if < 100ms since last sent)
       - `fun stop()` — release camera and ImageReader
       - Uses Camera2 API (not CameraX) to match the existing app's working pattern on RK3588. This is the same Camera2 approach from the old app that sends continuous JPEG frames for Dolphin speech separation on the server.

       IMPORTANT: This is separate from SelfieCapture (which is user-triggered photo). VideoStreamManager runs continuously in the background during conversation for the server's vision pipeline (face detection, lip frames for Dolphin).

         - Call `videoStreamManager.start(context)` to begin sending continuous 0x02 JPEG frames
         - In `onCleared()`: call `videoStreamManager.stop()`

         - `private fun resetSilenceTimer()` — cancels existing silenceJob, launches new coroutine: `delay(30_000L)` then emit `UiEvent.NavigateTo(Screen.Home)` and call `clearTranscript()`
         - Call `resetSilenceTimer()` on EVERY incoming WebSocket event (binary or JSON) — any server activity means the session is alive
         - Start the timer in init block

       - In `init` block, launch coroutine to collect `wsRepo.events`:
         - On ANY event: call `resetSilenceTimer()`
         - `JsonMessage(type="transcript")` → parse payload JSON for "text" field, append `ChatMessage(id=UUID, text=text, isUser=true)` to transcript
         - `JsonMessage(type="response")` → parse payload JSON for "text" field, append `ChatMessage(id=UUID, text=text, isUser=false)` to transcript

         - `JsonMessage(type="tts_control")` → forward to ttsPlayer (stop/start commands)
         - `BinaryFrame` where bytes[0]==0x05 → `ttsPlayer.handleBinaryFrame(bytes)` (strip type byte or pass full array per existing TtsAudioPlayer API)

         - Set showFeedback=false
         - Emit `UiEvent.NavigateTo(Screen.Home)` — auto-return even if skipped
         - Call `clearTranscript()`
