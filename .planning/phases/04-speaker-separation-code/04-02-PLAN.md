Add a `_doa_score_for_face` method to EngagementDetector that computes angular proximity between a face's position and the DOA direction:

```python
def _doa_score_for_face(self, track: FaceTrack, frame_width: int = 640,
                         camera_fov_deg: float = 60.0) -> float:
    """Return DOA alignment multiplier (1.0 = perfect, 0.5 = opposed)."""
    if self._last_doa_angle is None:
        return 1.0  # No DOA data — no penalty
    # FaceTrack.bbox is (x, y, w, h)
    x, _y, w, _h = track.bbox
    face_center_x = x + w / 2
    # Map pixel X to camera angle: center=0, left=negative, right=positive
    normalized = (face_center_x / frame_width) - 0.5  # [-0.5, 0.5]
    face_angle_deg = normalized * camera_fov_deg
    # DOA: 0=front, negative=left, positive=right
    angular_distance = abs(face_angle_deg - self._last_doa_angle)
    return max(0.5, 1.0 - angular_distance / 90.0)
```

Update `_select_primary_user` to use `_doa_score_for_face` instead of the flat 1.2x bonus:
